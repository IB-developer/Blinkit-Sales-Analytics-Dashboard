# 🛒 Blinkit Sales Analysis Dashboard

A comprehensive **Power BI Dashboard** built to analyze Blinkit's grocery sales performance across different outlet types, item categories, locations, and years. This dashboard provides interactive insights that help understand sales trends, customer preferences, and outlet performance.

---

## 📌 Project Overview

This project demonstrates end-to-end data analysis using **Power BI**. It transforms raw Blinkit grocery sales data into an interactive dashboard that enables users to monitor KPIs and make data-driven decisions.

---

## 🚀 Features

- Interactive Power BI Dashboard
- Dynamic KPI Cards
- Multiple Slicers for filtering
- Sales Trend Analysis
- Outlet Performance Analysis
- Item Category Analysis
- Customer Rating Analysis
- Responsive Visualizations

---

## 📊 Dashboard KPIs

The dashboard tracks the following business metrics:

- 💰 Total Sales
- 📦 Number of Items
- 💵 Average Sales
- ⭐ Average Rating

---

## 📈 Dashboard Visualizations

### 1. KPI Cards
Displays overall business performance using:

- Total Sales
- Average Sales
- Number of Items
- Average Rating

---

### 2. Sales by Item Fat Content

**Visualization:** Donut Chart

Purpose:
- Compare sales generated by different fat content categories.

---

### 3. Sales by Outlet Size

**Visualization:** Donut Chart

Purpose:
- Analyze contribution of each outlet size toward total sales.

---

### 4. Sales by Outlet Location Type

**Visualization:** Clustered Bar Chart

Purpose:
- Compare outlet performance across different locations.
- Split by Item Fat Content.

---

### 5. Sales by Item Type

**Visualization:** Bar Chart

Purpose:
- Identify the highest selling product categories.

---

### 6. Outlet Establishment Trend

**Visualization:** Line Chart

Purpose:
- Analyze how sales vary with outlet establishment year.

---

### 7. Outlet Location Sales Funnel

**Visualization:** Funnel Chart

Purpose:
- Compare sales contribution across outlet locations.

---

### 8. Outlet Performance Summary

**Visualization:** Pivot Table

Includes:

- Total Sales
- Number of Items
- Average Sales
- Average Rating
- Item Visibility

Grouped by:

- Outlet Type
